# Arrow Packer for Agibot Digit Challenge Dataset

This toolkit provides scripts for Packing and Visualizing simulation data for the Agibot Digit Challenge.

## Quick Start

### 1. Data Packing

Converts raw simulation data into transformed Arrow format. Supports parallel processing across multiple jobs.

#### Run Command

```python
python3 robo_orchard_lab/dataset/agibot_digit/packer/arrow_pack_challenge.py \
    --output_dir data/example_agibot_digit_challenge/ \
    --input_dir /horizon-bucket/robot_lab2/users/shujie.luo/data/AgiBotWorldChallenge-2025/Manipulation-SimData \
    --task_name pack_in_the_supermarket \
    --robot_name G1_omnipicker \
    --urdf_path /horizon-bucket/robot_lab/users/xuewu.lin/urdf/G1_omnipicker.urdf \
    --num_jobs 100 \
    --job_idx 0 \
    --storage_feature_type lite \
    --force_overwrite
```

#### Key Arguments
- `--num_jobs`: Total number of parallel jobs for data slicing.
- `--job_idx`: Current job index (from 0 to n-1).
- `--storage_feature_type`: Set to `lite` to optimize storage by stripping non-essential fields.

### 2. Data Visualization

Quickly verify Arrow data quality by generating videos with joint and EE pose.

#### Run Command

```
python3 robo_orchard_lab/dataset/agibot_digit/packer/arrow_viz_challenge.py \
    --output_dir data/example_agibot_digit_challenge/visualize \
    --input_dir data/example_agibot_digit_challenge/part-00000-of-00100 \
    --urdf_path /horizon-bucket/robot_lab/users/xuewu.lin/urdf/G1_omnipicker.urdf \
    --episode_index 0 \
    --fps 10
```

#### Key Arguments

- `--input_dir`: Path to a specific part-xxxxx directory generated by the packer.
- `--episode_index`: The specific episode to visualize.
